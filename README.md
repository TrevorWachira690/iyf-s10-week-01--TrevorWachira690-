## Project Description
I built a personal portfolio website that shows my personal information, how to contact me and the projects I have built. I built it to gain more understanding about HTML and CSS.

## Technologies Used
- HTML
- CSS
- JavaScript
- VScode
- Git
- Github

## Features
- [Home page](https://trevorwachira690.github.io/iyf-s10-week-01--TrevorWachira690-/index.html)- The website has a webpage which briefly introduces you to the portfolio whie also giving you redirections to other sections.
- [About me page](https://trevorwachira690.github.io/iyf-s10-week-01--TrevorWachira690-/about.html)- This is the page that gives you a full description about what I am doing, skills I am building, education and hobbies.
- [Projects page](https://trevorwachira690.github.io/iyf-s10-week-01--TrevorWachira690-/projects.html)- This is the section which shows you some of the projects that I have built with visible images.
- [Contact page](https://trevorwachira690.github.io/iyf-s10-week-01--TrevorWachira690-/contact.html)- This is the section of the portfolio that shows you how you can reach out to me via email, github or linkedin

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
   OR
   Run `npm install` then `npm start`

## Lessons Learned
I learnt how to use CSS efficiently in creating my webpages and I learnt how to use images on my web pages without loosing its meta content.

## Challenges Faced
1. **CSS styling issues**- I faced the issue of having to decide on which CSS styling method to use so I decided to stick with the internal styling method.
2. **Pushing errors**-While pushing my new changes I realized I made changes in the main repository on github and forgot to pull those changes, leading me to restructure my work.
